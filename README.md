# Airport Code Quiz

日本の空港3レターコード（IATAコード）を学ぶためのクイズゲームです。

## 機能
- **2つのモード**:
  - Normal: コードを見て空港名を当てる
  - Hard: 空港名を見てコードを当てる
- **タイマー機能**: 1問ごとの制限時間を設定可能（デフォルト10秒）
- **レスポンシブデザイン**: PC/スマホ両対応
- **PWA対応**: なし（シンプルな静的HTMLサイト）

## 実行方法
`index.html`をブラウザで開くだけで動作します。

## デプロイ
GitHub Pages等で簡単に公開可能です。
